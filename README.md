# RedQueen 红皇后投资助手系统

## 项目概述

RedQueen 是一个基于量化规则的A股市场异动识别系统，通过14条写死的量化规则对全市场个股进行扫描，捕获触发异动的个股，并提供行业风险分析。系统严格遵循零预测导向原则，所有输出100%基于已发生的历史数据。

### 核心功能

- **全市场扫描**：基于14条量化规则对全市场个股进行扫描
- **异动识别**：捕获触发≥1条规则的异动个股
- **AI行业映射**：通过大模型实现个股到行业的映射
- **行业风险分析**：提取行业共性风险，标注信源与发布日期
- **数据可视化**：提供异动个股列表和详情展示
- **清单导出**：支持导出异动个股清单为CSV格式

## 技术栈

- **后端**：Python 3.10+、FastAPI、SQLAlchemy、Pydantic
- **前端**：React + Vite + TypeScript + Ant Design + Axios + Recharts
- **数据库**：MySQL 8.0+
- **AI**：豆包大模型API

## 安装指南

### 前置条件

- Python 3.10+
- Node.js 16+
- MySQL 8.0+
- 豆包大模型API Key

### 后端安装

1. 进入后端目录
   ```bash
   cd redqueen/backend
   ```

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

3. 配置环境变量
   创建 `.env` 文件，填入以下内容：
   ```
   # 数据库配置
   DB_HOST=localhost
   DB_PORT=3306
   DB_USER=root
   DB_PASSWORD=your_password
   DB_NAME=stock_db

   # 豆包大模型API配置
   DOUBAO_API_KEY=your_api_key
   DOUBAO_API_ENDPOINT=https://ark.cn-beijing.volces.com/api/v3/responses
   ```

4. 初始化数据库
   执行 SQL 脚本创建表结构：
   ```bash
   mysql -u root -p stock_db < src/models/init.sql
   ```

5. 启动后端服务
   ```bash
   bash start.sh
   ```

### 前端安装

1. 进入前端目录
   ```bash
   cd redqueen/frontend
   ```

2. 安装依赖
   ```bash
   npm install
   ```

3. 启动前端服务
   ```bash
   npm run dev
   ```

## Docker部署

### 构建镜像

1. 构建后端镜像
   ```bash
   cd redqueen/backend
   docker build -t redqueen-backend .
   ```

2. 构建前端镜像
   ```bash
   cd redqueen/frontend
   docker build -t redqueen-frontend .
   ```

### 运行容器

使用 `docker-compose` 运行整个应用：

```bash
cd redqueen
docker-compose up -d
```

## 使用说明

### 手动触发扫描

1. 打开前端页面（默认 http://localhost:5173）
2. 点击 "手动触发扫描" 按钮
3. 等待扫描完成（全市场扫描约需要5分钟）
4. 扫描完成后，页面会自动刷新，显示异动个股列表

### 查看异动个股

1. 在列表页查看所有异动个股
2. 点击股票代码进入详情页
3. 在详情页查看触发的规则详情和行业风险分析

### 导出清单

1. 在列表页选择日期
2. 点击 "导出清单" 按钮
3. 下载CSV格式的异动个股清单

## API文档

后端API提供以下接口：

- **POST /api/scan/trigger** - 触发全市场扫描
- **GET /api/scan/status/{task_id}** - 获取扫描任务状态
- **GET /api/anomaly/stocks** - 获取异动个股列表
- **GET /api/anomaly/stock/{stock_code}** - 获取异动个股详情
- **GET /api/anomaly/export** - 导出异动个股清单
- **GET /api/health** - 健康检查

详细的API文档可通过访问 http://localhost:8000/docs 查看。

## 量化规则

系统基于以下14条量化规则进行异动识别：

1. **均线交叉趋势异动** - 均线金叉 + 量能确认
2. **趋势线突破异动** - 上升趋势线企稳突破
3. **道氏高低点趋势异动** - 更高低点 + 更高高点
4. **MACD趋势背离异动** - 底背离 + 金叉
5. **布林带通道突破异动** - 收敛后向上突破
6. **分位数回归趋势异动** - 上轨分位向上拐头
7. **量价背离趋势异动** - 价量底背离
8. **OBV能量潮趋势异动** - OBV先行新高
9. **主力资金流趋势异动** - 持续净流入
10. **换手率趋势异动** - 低位放量换手
11. **ATR波动率异动** - 波动率抬升 + 价格向上
12. **波动率收敛-发散异动** - 收敛后向上发散
13. **振幅异动趋势识别** - 振幅放大 + 股价上行

## 合规声明

本工具仅为A股市场已收盘数据的客观扫描结果，100%基于已发生的事实数据整理，不构成任何投资建议，不预测个股未来股价走势，不提供任何买卖指导。投资有风险，入市需谨慎。

## 注意事项

1. 系统需要连接到MySQL数据库，确保数据库服务正常运行
2. 系统需要有效的豆包大模型API Key，用于行业映射和风险分析
3. 全市场扫描可能需要较长时间（约5分钟），请耐心等待
4. 系统仅作为投资辅助工具，不构成任何投资建议

## 故障排除

1. **数据库连接失败**：检查数据库配置和服务状态
2. **API调用失败**：检查网络连接和API Key是否正确
3. **扫描任务失败**：查看任务状态和错误信息
4. **前端页面无法加载**：检查前端服务是否正常运行
